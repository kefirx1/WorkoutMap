# WorkoutMap v0.7.2

## About project
UI language - polish

Mobile application used for saving and monitoring workout progress. The app 
allows you to create your own training plans and track your weight changes.

## Tech stack

- Kotlin
- Android SDK
- Jetpack Compose
- MVVM (View Mode, Room database, Firebase)
- Dagger/Hilt
- Coroutines
- Gson
